Simple Tree Menu - Example

The tree menus can be nested to any depth.

You can of course have more than one tree menu on a page.

An example is given of an empty node, News, which is rendered in grey via the CSS class .EmptyNode overriding the normal .Node class. If you ignore this, empty nodes will work as normal.

Examples of managing the tree nodes:

  1. Expand all nodes
  2. Close all nodes
  3. Expand to node 'Bike Radar'
  4. Serialise state to local storage
    5. (Open some nodes, then click this, then refresh the page or open it in a new tab to see the tree redraw itself to the saved state)
